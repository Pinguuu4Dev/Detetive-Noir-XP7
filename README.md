# Como utilizar o projeto do Detetive Noir na Godot
Quando você abrir o arquivo do jogo na Godot, terá 5 abas nesta ordem da esquerda para a direita: 
1. 2D - 3D - Script - Game - AssetLib - Dialogic

2D: Visão 2D do projeto | 3D: Visão 3D do projeto.
Script: Janela de edição de Scripts através da programação
Game: Onde o jogo irá rodar ao apertar o símbolo de Play no canto superior direito, indicado por uma seta para a direita
AssetLib: Serve para adquirir plug-ins oficiais da Godot
Dialogic: Aba que tem a janela de edição dos elementos de Visual Novel do jogo, como Timelines, Personagens, Estilos de caixa de dialógo, etc.
